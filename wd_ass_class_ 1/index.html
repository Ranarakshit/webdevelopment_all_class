How to View HTML Source?

Have you ever seen a Web page and wondered "Hey! How did it they do that?"

View HTML source Code:

right -click in an HTML page and select "View Page Source"(in Chrome) or " "View Source" (in Edge), or similar in other browsers. This will open a window containning the HTML source code of the page.

inspect an HTML Element:

Right-click on an element (or a blank area), and choose "Inspect" or "Inspect Element" to see what elements are made up (you dwill see both the HTMl and the CSS). You can also edit the HTML or CSS on-the-fly in the elements or Styles panel that opens.
